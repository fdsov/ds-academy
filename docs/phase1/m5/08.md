# M5.8 · Broadcasting

### Интуиция

Broadcasting — это правила, по которым NumPy «растягивает» массивы разной формы, чтобы операция стала поэлементной, не копируя данные физически. Простейший пример: `array * 2` умножает каждый элемент на 2 — скаляр «растягивается» на всю форму. То же работает и между массивами: вектор может растянуться вдоль матрицы.

Практический смысл: нормировать матрицу признаков по столбцам, вычесть среднее из каждой строки, применить курс валют к таблице сумм — всё без циклов.

### Формально

Две формы совместимы, если, выравнивая их **справа налево**, по каждой оси выполняется хотя бы одно: размеры равны, либо один из них равен 1. Ось размера 1 «растягивается» до размера партнёра. Недостающие оси слева достраиваются как 1.

$$
(3, 4) \;\text{и}\; (4,) \rightarrow (3, 4) \;\text{и}\; (1, 4) \rightarrow (3, 4)
$$

$$
(3, 1) \;\text{и}\; (1, 4) \rightarrow (3, 4)
$$

Несовместимо, например, `(3,)` и `(4,)` — размеры не равны и ни один не 1.

```python
# нормировка z-score по столбцам матрицы признаков
X = rng.normal(size=(1000, 5))
mu = X.mean(axis=0)        # shape (5,) — среднее по каждому столбцу
sigma = X.std(axis=0)      # shape (5,)
X_norm = (X - mu) / sigma  # (1000,5) - (5,) -> broadcast по строкам

# вычесть среднее ИЗ КАЖДОЙ строки нужен столбец (n,1)
row_mean = X.mean(axis=1, keepdims=True)  # shape (1000,1)
X_centered = X - row_mean                 # broadcast по столбцам
```

Обрати внимание на `keepdims=True` — он сохраняет ось как размер 1 вместо её удаления, и тогда broadcasting срабатывает в нужном направлении. Без него `X.mean(axis=1)` даст shape `(1000,)`, который выровняется справа к столбцам и сломает логику.

!!! question "Проверь себя"

    1. Сформулируй правило совместимости форм при broadcasting.
    2. Совместимы ли `(3, 4)` и `(4,)`? А `(3,)` и `(4,)`?
    3. Зачем нужен `keepdims=True` при вычитании среднего из каждой строки матрицы?

??? success "Ответы"

    1. Выравнивая формы справа налево, по каждой оси либо размеры равны, либо один из них равен 1 (тогда он растягивается до размера партнёра); недостающие оси слева достраиваются как 1.
    2. `(3, 4)` и `(4,)` совместимы → `(3, 4)`. `(3,)` и `(4,)` несовместимы: размеры не равны и ни один не равен 1.
    3. Он сохраняет схлопнутую ось как размер 1: `X.mean(axis=1, keepdims=True)` даёт `(n, 1)`, который растягивается по столбцам. Без него результат `(n,)` выровняется справа к столбцам и посчитает не то.
