# M5.19 · Ресурсы

- NumPy Documentation (numpy.org) — официальные «Absolute Beginner's Guide» и «NumPy fundamentals», обновлены под версию 2.x.
- Jake VanderPlas, «Python Data Science Handbook», 2-е издание — глава по NumPy остаётся эталонным введением; доступна бесплатно онлайн.
- Wes McKinney, «Python for Data Analysis», 3rd ed. (2022) — глава 4 про NumPy от автора pandas, мост к табличным данным.
- NEP 19 и документация по `Generator` — первоисточник по современному random API и почему legacy seed устарел.
- «From Python to NumPy» Нико Руже (Nicolas Rougier) — глубокое бесплатное руководство именно про векторизацию и broadcasting с разбором памяти.
- 100 numpy exercises (репозиторий rougier/numpy-100) — набор задач по нарастающей для самопроверки.
- Scientific Python Lectures (ранее SciPy Lecture Notes), раздел NumPy — академически строгое изложение с упором на strides и память.
