# M4.18 · Ресурсы

- "SQL for Data Scientists" — Renée Teate. Практичный мост от SQL к задачам аналитики данных.
- "Learning SQL" — Alan Beaulieu (O'Reilly). Прочная база по реляционной модели и JOIN.
- PostgreSQL Documentation — раздел Window Functions и Performance Tips. Первоисточник, бесплатно и точно.
- DuckDB Documentation и блог — современные паттерны аналитики, чтение Parquet, интеграция с Python.
- "Practical SQL" — Anthony DeBarros. Сильный разбор работы с датами, агрегатами и реальными данными.
- Use The Index, Luke (use-the-index-luke.com) — Markus Winand. Лучший бесплатный ресурс по индексам и планам запросов.
- Mode Analytics SQL Tutorial и SQLBolt — интерактивная практика для закрепления.
- "Designing Data-Intensive Applications" — Martin Kleppmann. Для понимания, что под капотом у баз (research-level контекст).
