# M4.2 · Что ты освоишь

- [ ] Объяснять реляционную модель: таблицы, первичные и внешние ключи, связи один-ко-многим
- [ ] Писать запросы с SELECT / WHERE / GROUP BY / HAVING / ORDER BY и понимать порядок их выполнения
- [ ] Соединять таблицы всеми видами JOIN и осознанно выбирать нужный тип
- [ ] Строить когорты, retention и нарастающие итоги через оконные функции
- [ ] Корректно обращаться с NULL и не попадать в три классические ловушки
- [ ] Читать план запроса через EXPLAIN и понимать, почему запрос медленный
- [ ] Работать в двух диалектах — PostgreSQL для продакшена и DuckDB для аналитики
- [ ] Считать продуктовые метрики: DAU, retention, депозитные когорты — на чистом SQL
