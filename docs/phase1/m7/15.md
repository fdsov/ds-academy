# M7.15 · Ресурсы

- Hadley Wickham, "Tidy Data" (Journal of Statistical Software) — каноническая статья о том, какими должны быть чистые данные.
- Donald Rubin, "Inference and Missing Data" (Biometrika) — первоисточник классификации MCAR/MAR/MNAR.
- Stef van Buuren, "Flexible Imputation of Missing Data", 2nd ed. — глубоко и практично про импутацию и MICE.
- Документация Pandera и Pydantic v2 — основной инструмент валидации, читать с примерами.
- Документация Polars (User Guide) и Apache Arrow/Parquet — стандарт обработки и хранения 2026.
- DAMA-DMBOK, главы по Data Quality — индустриальный канон шести измерений качества.
- "Fundamentals of Data Engineering" (Reis, Housley, O'Reilly) — источники, форматы, слои bronze/silver/gold в общей картине.
- Документация httpx и руководства по корректной работе с REST API (пагинация, retry, backoff).
