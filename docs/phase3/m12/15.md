# M12.15 · Ресурсы

- Edward Tufte. The Visual Display of Quantitative Information — первоисточник data-ink ratio и chartjunk, обязательное чтение профессии.
- Leland Wilkinson. The Grammar of Graphics — теоретический фундамент всех современных систем визуализации.
- Cole Nussbaumer Knaflic. Storytelling with Data — практический стандарт по дата-нарративу для бизнес-аудитории.
- Claus O. Wilke. Fundamentals of Data Visualization — бесплатно онлайн, лучший современный разбор «какой график когда» с упором на честность.
- Cleveland, McGill (1984). Graphical Perception — статья, заложившая иерархию точности восприятия каналов.
- Документация seaborn (раздел objects interface) и altair (Vega-Lite 6) — актуальный на 2026 справочник по грамматике графики в Python.
- ColorBrewer 2.0 и палитры viridis — практический инструмент выбора colorblind-safe палитр под тип данных.
