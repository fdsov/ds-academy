# M12.10 · Дашборды: коротко

Когда график нужен не разово, а как живой интерфейс к обновляемым данным, — это дашборд.

- Streamlit — собираешь дата-приложение на чистом Python за часы; идеален для внутренних инструментов аналитика и быстрых демо моделей. Не BI-платформа, а способ обернуть свой скрипт в UI.
- Apache Superset — полноценная open-source BI поверх SQL-источников: дашборды, SQL Lab, разграничение доступа. Для продуктовой аналитики на уровне команды/компании.
- Metabase — самый низкий порог входа, «вопросы к данным» без SQL, быстрый старт для нетехнических коллег.

Правило дашборда то же, что и графика: каждый виджет отвечает на конкретный вопрос конкретной аудитории. Дашборд из 30 графиков «на всякий случай» — не инструмент, а свалка, в которую никто не смотрит.
