# M12.5 · Типы графиков: какой вопрос — такая геометрия

Не «какой график красивее», а «на какой вопрос про данные я отвечаю». Пять базовых семей вопросов.

### Распределение: как устроена одна переменная

Вопрос «как разбросаны депозиты / время сессии / возраст игроков».

- Гистограмма — делит ось на бины и считает частоту. Простая, но критично зависит от ширины бина: слишком широкие прячут структуру (например, бимодальность — два пика: мелкие и киты), слишком узкие создают «зубчатый» шум. Всегда пробуй несколько значений.
- Boxplot (ящик с усами) — показывает медиану, квартили $Q_1, Q_3$ и выбросы. Компактен, отлично для сравнения распределений между группами. Но прячет форму: бимодальное и равномерное распределения могут дать одинаковый ящик. Межквартильный размах $IQR = Q_3 - Q_1$, усы обычно до $1.5 \cdot IQR$, дальше — точки-выбросы.
- Violin plot — boxplot плюс зеркальная оценка плотности (KDE) по бокам. Возвращает форму, которую прячет ящик: сразу видна бимодальность. Платишь сглаживанием — на малых выборках KDE врёт.
- ECDF (эмпирическая функция распределения) — для каждого значения $x$ показывает долю наблюдений $\le x$:

$$
\hat{F}_n(x) = \frac{1}{n} \sum_{i=1}^{n} \mathbf{1}[X_i \le x]
$$

  ECDF — самый честный график распределения: не требует выбора бина, не сглаживает, читается как «у скольких процентов игроков депозит ниже порога». Профессионалы используют ECDF чаще гистограмм, особенно для сравнения двух групп (две монотонные кривые сравниваются глазом легко).

!!! tip "Гистограмма для презентации, ECDF для анализа"

    Стейкхолдеру показывай гистограмму или violin — они интуитивны. Сам анализируй на ECDF — она не обманывает выбором бина. И именно расстояние между двумя ECDF лежит в основе теста Колмогорова-Смирнова, которым проверяют, отличаются ли распределения двух A/B-групп.

### Связь: как две переменные зависят друг от друга

Scatter plot (точечный) — каждая точка наблюдение, $x$ и $y$ — две переменные. База для поиска корреляции, кластеров, выбросов. Вопрос «связан ли размер первого депозита с LTV». При больших $n$ точки сливаются в чёрное пятно (overplotting) — лечится прозрачностью (alpha), гексбином или 2D-плотностью. Добавь линию тренда (регрессия или LOESS), чтобы показать форму связи.

### Динамика: как переменная меняется во времени

Line plot (линейный) — $x$ это время, линия соединяет точки, подчёркивая непрерывность и тренд. Депозиты по дням, DAU по неделям, GGR по месяцам. Линию используй только когда по $x$ есть осмысленный порядок (обычно время); соединять линией категории — ошибка.

### Сравнение: как отличаются категории

Bar plot (столбчатый) — длина столбца кодирует величину по категории. Лучший канал восприятия после положения. Сравнение GGR по странам, конверсии по лендингам. Правило, которое нарушают постоянно: у bar-графика ось $y$ обязана начинаться с нуля — длина столбца и есть величина, обрезка делает её ложью. Сортируй столбцы по значению, а не по алфавиту, если нет естественного порядка.

### Состав: из чего собрано целое

- Stacked bar / stacked area — части внутри столбца или площади. Хорошо видна сумма и крупнейшая компонента (она лежит на базовой линии). Плохо сравниваются средние сегменты — у них «плавающее» основание.
- Круговая диаграмма (pie) — состав через углы. Работает только для 2-3 категорий с явно разными долями. Для большего — bar-график долей почти всегда лучше.

### Корреляции: структура многих переменных сразу

Heatmap (тепловая карта) — матрица, где цвет ячейки кодирует величину. Два главных применения: матрица корреляций между признаками (диагностика мультиколлинеарности перед регрессией) и когортная таблица retention (строки — когорты по дате регистрации, столбцы — возраст когорты в днях, цвет — доля вернувшихся). Когортная heatmap — один из самых мощных графиков продуктовой аналитики: за один взгляд видно и как меняется качество приходящих когорт (читаешь сверху вниз), и как игроки отваливаются со временем (читаешь слева направо).

!!! question "Проверь себя"

    1. Под каждый из пяти типов вопросов (распределение, связь, динамика, сравнение, состав) назови подходящую геометрию.
    2. Почему ECDF считается самым честным графиком распределения по сравнению с гистограммой?
    3. Что прячет boxplot и какой график это возвращает?
    4. Когортная retention-heatmap: что читается, если двигаться по ней сверху вниз, и что — слева направо?

??? success "Ответы"

    1. Распределение — гистограмма/boxplot/violin/ECDF; связь — scatter; динамика — line; сравнение — bar; состав — stacked bar/area или (для 2-3 долей) pie.
    2. ECDF не требует выбора ширины бина и не сглаживает: она показывает точную долю наблюдений ниже порога, поэтому не искажает форму произвольным параметром.
    3. Boxplot прячет форму распределения (бимодальное и равномерное могут дать одинаковый ящик); форму возвращает violin plot за счёт оценки плотности (KDE).
    4. Сверху вниз — как меняется качество приходящих когорт (по дате регистрации); слева направо — как игроки отваливаются с возрастом когорты.

!!! example "Мини-задание"

    Возьми любой числовой столбец (время сессии, депозит, возраст). Построй для него boxplot и violin рядом. В двух строках зафиксируй, какую особенность формы (например, второй пик) видно на violin, но не на boxplot.
