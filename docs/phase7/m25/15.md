# M25.15 · Ресурсы

- S. Keshav, "How to Read a Paper" — каноническая статья о трёхпроходном методе, 3 страницы, читается за вечер.
- Joelle Pineau et al., "Improving Reproducibility in Machine Learning Research" (NeurIPS Reproducibility Program) — first-hand про кризис воспроизводимости и чек-листы.
- Sayash Kapoor, Arvind Narayanan, "Leakage and the Reproducibility Crisis in ML-based Science" (2023) — систематика утечек данных, обязательное чтение.
- "AI Snake Oil" (Kapoor, Narayanan, 2024) — как отличать реальный прогресс от хайпа, на стыке науки и индустрии.
- Papers with Code (paperswithcode.com) — статьи, код, лидерборды; главный практический инструмент.
- OpenReview (openreview.net) — открытые рецензии ICLR; читай rebuttal-дискуссии как тренажёр критики.
- Zachary Lipton, Jacob Steinhardt, "Troubling Trends in Machine Learning Scholarship" — про честность письма и типичные грехи статей.
- The Turing Way (онлайн-руководство) — practical guide по воспроизводимым исследованиям и данным.
