# M28.7 · NLP, CV, recsys — альтернативные вертикали

Кратко, чтобы видеть карту. NLP (обработка языка) в 2026 почти полностью поглощён LLM: задачи — RAG, агенты, классификация тикетов, анализ тональности отзывов, модерация чата. CV (компьютерное зрение) — детекция объектов, OCR документов (KYC-верификация в гемблинге — распознавание паспортов), биометрия, медицинская диагностика. Recsys (рекомендательные системы) — ядро монетизации в e-commerce, стриминге и iGaming (какую игру показать игроку): коллаборативная фильтрация, матричная факторизация, two-tower нейросети, sequence-модели. Эти вертикали — полноценные альтернативы time series/quant/gambling. Выбор между ними возвращает к трём фильтрам из начала: преимущество, деньги, интерес.
