# M22.15 · Ресурсы

- Chip Huyen, «Designing Machine Learning Systems» (O'Reilly) — лучшая книга о ML в продакшене целиком, от данных до мониторинга.
- D. Sculley et al., «Hidden Technical Debt in Machine Learning Systems» (NeurIPS 2015) — фундаментальная статья о том, почему ML-системы коварнее обычного софта.
- Google Cloud, «MLOps: Continuous delivery and automation pipelines in machine learning» — каноничная статья про уровни зрелости MLOps и CT.
- Документация MLflow и DVC — рабочие референсы по трекингу, registry и версионированию данных.
- Документация Evidently и NannyML — практика мониторинга дрейфа и качества с отложенными labels.
- Документация FastAPI — официальное руководство по сервингу, включая асинхронность и продакшен-деплой.
- Andriy Burkov, «Machine Learning Engineering» — инженерный взгляд на полный цикл, дополняет Huyen.
- Курс «Made With ML» (Goku Mohandas) — практичный открытый курс по MLOps, актуализируется под современный стек.
