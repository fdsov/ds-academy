# M24.2 · Что ты освоишь

- [ ] Переводить precision/recall и ошибки модели в деньги через матрицу стоимости (cost-sensitive learning)
- [ ] Выбирать порог классификатора, максимизирующий прибыль, а не F1
- [ ] Считать unit-экономику (LTV, CAC, payback, retention) и связывать модель с выручкой
- [ ] Проектировать прод-эксперимент: online A/B, holdout, shadow mode, постепенный rollout
- [ ] Оценивать инкрементальный (causal) эффект модели, а не наблюдаемую корреляцию
- [ ] Применять uplift-моделирование для таргетинга: кого трогать, а кого нет
- [ ] Выстраивать продуктовые метрики (AARRR, North Star, guardrails) и приоритизировать через ICE/RICE
- [ ] Решать, когда модель НЕ нужна, и аргументировать это бизнесу языком решений
