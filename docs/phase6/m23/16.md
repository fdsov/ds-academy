# M23.16 · Ресурсы

- Joe Reis, Matt Housley. «Fundamentals of Data Engineering» (O'Reilly) — базовый учебник по жизненному циклу данных, без привязки к вендору.
- dbt Labs. Официальная документация и курс dbt Learn (learn.getdbt.com) — практический стандарт по трансформациям и тестам.
- DuckDB documentation (duckdb.org/docs) — короткая, отличная; раздел про чтение Parquet/Iceberg и оконные функции.
- Dagster University (courses.dagster.io) — бесплатный курс по asset-центричной оркестрации, интеграция с dbt.
- Apache Iceberg documentation (iceberg.apache.org) — концепции табличного формата, time travel, эволюция схемы.
- Martin Kleppmann. «Designing Data-Intensive Applications» — глубокая теория хранилищ, репликации, стриминга; для research-уровня.
- Книга «The Data Warehouse Toolkit» (Kimball) — классика dimensional modeling (факты/измерения), всё ещё актуальна для проектирования витрин.
- Блог и доклады по «Modern Data Stack 2026» (dbt Coalesce, Data Council talks) — чтобы держать руку на пульсе сдвигов индустрии.
