# M8.6 · Базовая комбинаторика

### Интуиция

Чтобы применить классическую вероятность, нужно уметь считать "сколько способов". Сколько комбинаций ставок, сколько способов выбрать 3 победителей из 50, сколько вариантов раздачи карт.

### Формально

Правило произведения: если первый выбор делается $n_1$ способами, второй $n_2$, то совместно — $n_1 \cdot n_2$.

Перестановки $n$ объектов: $n!$.

Размещения (порядок важен, выбираем $k$ из $n$):

$$
A_n^k = \frac{n!}{(n-k)!}
$$

Сочетания (порядок не важен):

$$
C_n^k = \binom{n}{k} = \frac{n!}{k!\,(n-k)!}
$$

Биномиальный коэффициент $\binom{n}{k}$ — сколькими способами получить $k$ успехов в $n$ испытаниях. Он напрямую войдёт в биномиальное распределение. Пример: из 50 игроков турнира выбрать 3 для приза без учёта порядка — $\binom{50}{3} = 19600$ способов.
