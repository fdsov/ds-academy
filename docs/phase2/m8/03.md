# M8.3 · Случайный эксперимент, исходы, события

### Интуиция

Случайный эксперимент — это любая ситуация, исход которой ты не можешь предсказать заранее, но можешь перечислить все варианты, чем она может закончиться. Бросок монеты, показ баннера пользователю (кликнет / не кликнет), сессия игрока (задепозитит / нет), число ставок за вечер. Заранее результат неизвестен, но множество возможных результатов очерчено.

### Формально

Вводим три объекта.

Пространство элементарных исходов $\Omega$ — множество всех взаимоисключающих результатов эксперимента. Для броска монеты $\Omega = \{\text{орёл}, \text{решка}\}$. Для числа депозитов за день $\Omega = \{0, 1, 2, \dots\}$.

Элементарный исход $\omega \in \Omega$ — один конкретный результат.

Событие $A$ — это подмножество $\Omega$, то есть набор исходов, которые нас интересуют как одно целое. "Игрок сделал хотя бы один депозит" — это событие $A = \{1, 2, 3, \dots\}$. Событие произошло, если реализовавшийся исход $\omega$ лежит в $A$.

Чтобы корректно говорить о вероятностях для бесконечных $\Omega$ (например, непрерывных), множество допустимых событий должно образовывать сигма-алгебру $\mathcal{F}$ — семейство подмножеств, замкнутое относительно дополнения и счётных объединений. Для дискретных задач можно без потерь брать все подмножества; сигма-алгебра становится критичной только для непрерывных пространств, где не каждому подмножеству можно осмысленно приписать вероятность.

!!! info "Зачем сигма-алгебра на практике"

    На уровне продуктовой аналитики ты почти никогда не пишешь сигма-алгебру руками. Но именно она — причина, по которой для непрерывной величины (размер чека) вероятность точного значения $P(X = 100.00) = 0$, а осмысленны только интервалы $P(90 \le X \le 110)$. Это не каприз математиков, а следствие устройства вероятностной меры.

!!! question "Проверь себя"

    1. Что такое пространство элементарных исходов $\Omega$ и чем элементарный исход отличается от события?
    2. Событие "игрок сделал хотя бы один депозит" при $\Omega = \{0, 1, 2, \dots\}$ — это какое подмножество?
    3. Почему для непрерывной величины вероятность точного значения равна нулю?

??? success "Ответы"

    1. $\Omega$ — множество всех взаимоисключающих результатов эксперимента. Элементарный исход $\omega$ — один конкретный результат; событие $A$ — подмножество $\Omega$, набор интересующих нас исходов как одно целое.
    2. $A = \{1, 2, 3, \dots\}$ — все исходы, кроме нуля.
    3. Из устройства вероятностной меры на непрерывном пространстве: масса размазана по интервалу как плотность, и осмысленны только интервалы $P(a \le X \le b)$, а у отдельной точки "ширины" нет, поэтому $P(X = x) = 0$.
