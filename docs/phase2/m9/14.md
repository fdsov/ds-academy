# M9.14 · Практика

### Быстрый квиз

Выбери вариант — ответ подсветится сразу.

```quiz
Куда направлена стрелка "модель — данные" в математической статистике?
[ ] От модели к данным: зная параметр, описываем поведение наблюдений
[x] От данных к модели: по наблюдениям восстанавливаем неизвестный параметр
[ ] В обе стороны одновременно, направление не определено
> Статистика — обратная задача к вероятности: даны данные, нужно реконструировать модель мира (параметр).
---
У тебя 10 миллионов строк в смещённой выборке. Что это даёт?
[ ] Снимает смещение, потому что объём данных огромен
[x] Очень точную оценку неправильной величины: объём бьёт дисперсию, но не смещение
[ ] Полностью репрезентативную картину популяции
> Объём борется со случайной ошибкой (дисперсией), а систематическое смещение отбора он не лечит.
---
Зачем в выборочной дисперсии $s^2$ стоит делитель $n-1$, а не $n$?
[x] Поправка Бесселя: деление на $n$ даёт смещённую оценку, систематически занижающую дисперсию
[ ] Чтобы оценка быстрее сходилась к истинному значению при росте $n$
[ ] Чтобы уменьшить дисперсию самой оценки до границы Рао — Крамера
> Мы уже потратили данные на $\bar{X}$, поэтому отклонения от $\bar{X}$ в среднем меньше; $n-1$ компенсирует это.
---
Чем стандартная ошибка $\text{SE}$ отличается от стандартного отклонения $\sigma$?
[ ] Это одно и то же, просто разные названия одной величины
[x] $\sigma$ — разброс отдельных наблюдений, $\text{SE}=\sigma/\sqrt{n}$ — разброс самой оценки по выборкам
[ ] $\text{SE}$ описывает разброс данных, а $\sigma$ — разброс среднего
> Игроки разнятся всегда ($\sigma$), но среднее по большой выборке стабильно — его дрожание и ловит $\text{SE}$.
---
Какая трактовка 95%-доверительного интервала корректна?
[ ] Вероятность того, что $\mu$ лежит внутри посчитанного интервала, равна 0.95
[x] При многократном повторении эксперимента 95% построенных интервалов накроют истинное $\mu$
[ ] 95% игроков имеют депозит внутри этого интервала
> 95% — свойство процедуры, а не конкретного интервала: $\mu$ фиксировано, случаен интервал.
---
В чём центральная идея bootstrap?
[x] Реальная популяция заменяется эмпирической: из выборки многократно пересэмплируют с возвращением
[ ] Из исходных данных генерируются новые, ранее не наблюдавшиеся значения
[ ] Аналитическая формула SE выводится для любой метрики автоматически
> Выборка считается лучшей моделью популяции; разброс пересчитанной метрики и есть оценка неопределённости.
---
Почему для редкой конверсии (1.5% на малом сегменте) нельзя брать Wald-интервал?
[ ] Wald всегда занижает оценку самой доли $\hat{p}$
[x] Он может уйти ниже нуля и иметь неверное покрытие; нужен Wilson или Agresti — Coull
[ ] Wald работает только для среднего, но не для пропорций в принципе
> На малых $\hat{p}$ или $n$ Wald даёт чушь (отрицательную границу); корректнее интервал Уилсона.
```

### Разминка

1. Сгенерируй 1000 значений из нормального распределения с известными $\mu, \sigma$. Посчитай $\bar{x}$, $s$, $\text{SE}$. Объясни словами разницу между $s$ и $\text{SE}$ на своих числах.
2. Построй 90%, 95% и 99% доверительные интервалы для среднего из пункта 1. Проследи, как меняется ширина, и объясни почему.
3. Для пропорции (конверсия $k=37$ из $n=500$) посчитай Wald- и Wilson-интервалы. Найди, где они расходятся сильнее всего, и сформулируй вывод.

### Основное

4. Реализуй bootstrap "руками" (без `scipy.stats.bootstrap`): функция принимает массив и статистику, возвращает SE и percentile-CI. Сравни результат со встроенным `stats.bootstrap` на медиане логнормальной выборки.
5. Выведи на бумаге MLE для распределения Пуассона (число ставок за сессию). Подтверди кодом на синтетике, что MLE совпадает с выборочным средним.
6. Эксперимент на состоятельность: для $n \in \{10, 100, 1000, 10000, 100000\}$ много раз оцени среднее и построй график, как сужается распределение оценки. Покажи закон $\sqrt{n}$ численно.
7. Эксперимент на покрытие: построй 1000 доверительных интервалов 95% для известного $\mu$ и проверь, что примерно 95% из них накрывают истинное значение. Это материализует корректную трактовку CI.

### Челлендж

8. Создай намеренно смещённую выборку депозитов (например, выкинь нижние 20% — имитация "видим только задепозитивших крупно"). Покажи, что увеличение объёма такой выборки НЕ приближает оценку к истинному $\mu$ полной популяции. Сформулируй, почему.
9. Сравни нормальный (CLT) и bootstrap доверительные интервалы для среднего сильно скошенного распределения при $n=30$ и $n=2000$. При каком $n$ они сходятся и почему расходились.

### Мини-проект

10. Возьми (или синтезируй) когортные данные по retention: для дней 1, 3, 7, 14, 30 — доли вернувшихся игроков. Для каждой точки построй bootstrap 95% доверительный интервал и Wilson-интервал, наложи их на retention-кривую как полосу неопределённости. Отдельно покажи, как survivorship меняет картину: посчитай retention D30 по всей исходной когорте против retention D30 только среди доживших до D14, и объясни разницу стейкхолдеру одним абзацем без жаргона. Результат — график retention-кривой с доверительной полосой плюс короткая записка с выводами.
