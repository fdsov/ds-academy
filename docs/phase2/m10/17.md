# M10.17 · Ресурсы

- Ron Kohavi, Diane Tang, Ya Xu. "Trustworthy Online Controlled Experiments" (Cambridge, 2020) — настольная книга по A/B в продукте, авторы из Microsoft/Airbnb.
- Georgi Georgiev. "Statistical Methods in Online A/B Testing" (2019) — последовательный анализ и реальные ловушки практики.
- Larry Wasserman. "All of Statistics" — строгий фундамент проверки гипотез и мощности.
- Microsoft ExP / статьи команды Kohavi об SRM, twyman's law и дисперсионной редукции (CUPED original paper, Deng et al., 2013).
- Evan Miller, "How Not To Run an A/B Test" и его онлайн-калькуляторы — короткое классическое эссе про peeking.
- Документация `statsmodels.stats` и `pingouin` — актуальные API расчётов мощности и тестов на 2026.
- Sequential / always-valid testing: статьи Howard et al. о confidence sequences и материалы по mSPRT (Optimizely, Eppo).
- Cameron Davidson-Pilon. "Bayesian Methods for Hackers" — практичное введение в байесовский подход с кодом.
