# M16.4 · Типы признаков

Прежде чем кодировать, нужно понимать, с чем имеешь дело. Тип признака диктует обработку.

- Числовые непрерывные: сумма депозита, коэффициент ставки, время сессии. Можно масштабировать, биннить, логарифмировать.
- Числовые дискретные счётчики: число ставок за день, число депозитов. Часто скошены, любят логарифм $\log(1+x)$.
- Категориальные номинальные: страна, платёжный метод, провайдер игры — порядка нет.
- Категориальные порядковые: VIP-уровень, грейд риска — порядок есть, его надо сохранить.
- Бинарные: верифицирован ли аккаунт, был ли FTD (first time deposit).
- Временные: дата регистрации, метка времени ставки — из них извлекают много производных.
- Текстовые: комментарий в поддержку, описание транзакции.
- Высококардинальные категории: id игры из тысяч вариантов, город — отдельная боль, one-hot тут взрывается.

Понимание типа — это уже половина решения о кодировании. Дальше разберём кодирование подробно.
