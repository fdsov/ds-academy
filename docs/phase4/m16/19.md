# M16.19 · Ресурсы

- Max Kuhn, Kjell Johnson. Feature Engineering and Selection (бесплатно на bookdown) — глубоко и строго про признаки и отбор.
- Alice Zheng, Amanda Casari. Feature Engineering for Machine Learning (O'Reilly) — практический справочник по преобразованиям.
- Bergstra, Bengio. Random Search for Hyper-Parameter Optimization (JMLR 2012) — почему random бьёт grid, первоисточник.
- Akiba et al. Optuna: A Next-generation Hyperparameter Optimization Framework (KDD 2019) + актуальная документация Optuna 4.x.
- Документация scikit-learn: разделы Pipelines, ColumnTransformer, TargetEncoder, Cross-validation (nested CV пример) — читать как первоисточник API 2026.
- Prokhorenkova et al. CatBoost: unbiased boosting with categorical features (NeurIPS 2018) — про ordered target statistics и борьбу с утечкой кодирования.
- Christoph Molnar. Interpretable Machine Learning — главы про permutation importance и SHAP для honest feature importance.
- Документация LightGBM и polars — нативная работа с категориями, пропусками и быстрые оконные агрегаты по логам.
