# M16.9 · Биннинг

Интуиция: разбить непрерывный признак на корзины. Возраст → группы, депозит → tiers «низкий/средний/кит». Иногда нелинейную зависимость проще выразить категориями, чем числом.

Формально два подхода. Равноширинный — отрезки одинаковой длины. Равночастотный (квантильный) — границы по квантилям, в каждой корзине поровну объектов; устойчивее к выбросам.

Когда полезно: для линейных моделей биннинг позволяет ловить нелинейность без явных полиномов. Для скоринговых карт (как в кредитном риске) биннинг — стандарт, потому что даёт монотонные интерпретируемые группы. Минус — теряешь информацию внутри корзины и вводишь произвольные границы. Деревьям биннинг обычно не нужен: они и так находят пороги. Не биннь бездумно — это потеря разрешающей способности.
