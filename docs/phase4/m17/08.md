# M17.8 · Интерпретируемость: глобальная против локальной

### Интуиция

Глобальная интерпретируемость отвечает на вопрос "как модель работает в целом, какие признаки в среднем важны?". Локальная — "почему модель приняла именно это решение для конкретного игрока?". Это разные вопросы. Продукт-менеджер хочет глобальную картину ("отток в основном про снижение частоты ставок"). Саппорт, объясняющий игроку блокировку, хочет локальную ("этот конкретный игрок помечен из-за вот этих трёх транзакций").

### Формально

- **Глобальная:** характеризует поведение функции $\hat{f}$ на всём распределении данных. Инструменты: коэффициенты линейных моделей, feature importance деревьев, permutation importance, глобальный SHAP summary, partial dependence.
- **Локальная:** объясняет одно предсказание $\hat{f}(x_i)$ для конкретного $x_i$. Инструменты: SHAP values для строки, LIME, локальные surrogate-модели.
