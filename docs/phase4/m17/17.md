# M17.17 · Ресурсы

- Christoph Molnar. Interpretable Machine Learning, 2-е изд. (онлайн, обновляется) — лучший единый источник по permutation importance, SHAP, PDP, ALE.
- Документация SHAP (shap.readthedocs.io) — актуальные API, TreeExplainer, примеры на табличных данных.
- scikit-learn User Guide, разделы Cross-validation, Model evaluation, Calibration, Permutation importance — каноничные определения и корректные примеры.
- Lundberg & Lee. A Unified Approach to Interpreting Model Predictions (2017) — первоисточник SHAP, для понимания значений Шепли.
- Hastie, Tibshirani, Friedman. The Elements of Statistical Learning — bias-variance, обобщающая ошибка, основа теории валидации.
- Kaggle. Data Leakage micro-course — короткий практичный материал с примерами реальных утечек.
- Документация Evidently AI — мониторинг data drift и distribution shift в продакшене на 2026.
- Статья "Common Pitfalls in the Interpretation of Coefficients of Linear Models" (scikit-learn examples) — подвохи коэффициентов и мультиколлинеарности.
