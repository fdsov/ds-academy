# M17.13 · Воспроизводимость

Result, который нельзя повторить, — не result. Фиксируй:

- **Seed** для всех источников случайности (`numpy`, `random`, сплиттеры, модель). Без этого числа пляшут от запуска к запуску.
- **Версии библиотек** — `requirements.txt`/`uv.lock`/`poetry.lock`. Поведение `sklearn`/`xgboost` меняется между мажорами.
- **Версия данных** — снапшот или хэш датасета. Данные меняются молча.
- **Конфиг эксперимента** — гиперпараметры, схема сплита, препроцессинг. Логируй через MLflow или подобное.

Если коллега не может повторить твой AUC одной командой — работа не закончена.
