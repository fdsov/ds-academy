# M13.5 · Базовая терминология

- **Признаки (features, $X$)** — входные переменные, описывающие объект. Для игрока: возраст аккаунта, число депозитов, средний чек, любимая игра, гео. Матрица признаков имеет форму $n \times d$ ($n$ объектов, $d$ признаков).
- **Таргет (target, label, $y$)** — то, что предсказываем. Для churn — бинарный флаг "ушёл за 30 дней".
- **Объект / наблюдение (sample)** — одна строка, один игрок.
- **Обучение (training, fit)** — процесс подгонки параметров модели под обучающие данные.
- **Инференс (inference, predict)** — применение обученной модели к новым данным для получения предсказаний.

Ключевое различие, которое путают новички: обучение происходит один раз (или периодически при переобучении), а инференс — постоянно, на каждом новом игроке. Всё, что модель "знает", зафиксировано на момент обучения.
