# M14.14 · Ресурсы

- Hastie, Tibshirani, Friedman. The Elements of Statistical Learning (2-е изд.) — библия классического ML, главы про линейные модели, регуляризацию, деревья и бустинг. Бесплатна на сайте Стэнфорда.
- James, Witten, Hastie, Tibshirani. An Introduction to Statistical Learning, Python edition (2-е изд., 2023) — мягкий вход в те же темы с кодом на Python, доступна бесплатно.
- Aurelien Geron. Hands-On Machine Learning with Scikit-Learn, Keras and TensorFlow (3-е изд.) — лучший практический мост от теории к коду, главы по ансамблям и бустингу обязательны.
- Christoph Molnar. Interpretable Machine Learning (онлайн, обновляемая) — глубоко про SHAP, permutation importance, ловушки интерпретации feature importance.
- Документация CatBoost, LightGBM, XGBoost — официальные tutorials по категориальным признакам, early stopping и тюнингу. Источник истины по актуальному API на 2026.
- SHAP documentation (Lundberg) — теория значений Шепли и рецепты применения TreeExplainer к бустингу.
- Курс Machine Learning от Andrew Ng (Coursera/DeepLearning.AI, обновлённая версия) — для интуиции по градиентному спуску, регуляризации и логистической регрессии с нуля.
- Kaggle Learn: Intro to Machine Learning и Feature Engineering — короткие практические треки, чтобы набить руку на реальных табличных данных.
