# M14.13 · Ты освоил это, когда

- Можешь объяснить десятилетнему ребёнку, почему Lasso «выбрасывает» лишние признаки, а Ridge только «приглушает» — через картинку ромба и круга, без формул.
- Берёшь коэффициент логистической регрессии и без подглядывания переводишь его в odds ratio и в фразу на языке продакта («каждый день простоя повышает шанс оттока на 5%»).
- Объясняешь коллеге, почему его модель с accuracy 99% на фроде бесполезна, и какие метрики надо смотреть вместо этого.
- Можешь на пальцах рассказать, почему Random Forest сильнее простого бэггинга, сведя всё к одному слову — декорреляция — и пояснив, откуда берётся потолок $\rho \sigma^2$.
- Видишь в чужом ноутбуке утечку данных (препроцессинг до сплита, признак из будущего) за полминуты и можешь объяснить, почему AUC врёт.
- Обоснованно выбираешь между XGBoost, LightGBM и CatBoost под конкретные данные, а не по привычке, и можешь назвать главное отличие каждого.
