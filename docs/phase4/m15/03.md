# M15.3 · Природа задачи без таргета

### Интуиция

В обучении с учителем ты как студент с решебником: решаешь задачу, заглядываешь в ответ, корректируешься. В обучении без учителя решебника нет вообще. Тебе дали кучу фотографий и сказали «разложи похожее к похожему» — но никто не сказал, по какому признаку похожее: по цвету фона, по выражению лица, по времени суток? Ты сам решаешь, что считать структурой.

Отсюда главное следствие: в unsupervised не бывает однозначно «правильного» ответа. Бывает ответ полезный или бесполезный для конкретной задачи. Два разных алгоритма дадут две разные, и обе по-своему валидные сегментации одних и тех же игроков.

### Формально

Пусть есть выборка $X = \{x_1, x_2, \ldots, x_n\}$, где каждый $x_i \in \mathbb{R}^d$ — вектор признаков объекта. В отличие от supervised, меток $y_i$ нет. Задача — найти отображение или структуру, которая описывает распределение $p(x)$ или его геометрию.

Три главных класса задач:

- Кластеризация: разбить $X$ на группы $C_1, \ldots, C_k$ так, чтобы внутри группы объекты были похожи, между группами — различны.
- Снижение размерности: найти отображение $f: \mathbb{R}^d \to \mathbb{R}^m$ при $m \ll d$, сохраняющее «важную» структуру.
- Поиск аномалий: найти объекты, для которых $p(x)$ мала, то есть нетипичные точки.

Все три объединяет отсутствие $y$ и, как следствие, проблема валидации — к ней вернёмся отдельно.

!!! question "Проверь себя"

    1. Чем принципиально отличается постановка задачи в unsupervised от supervised?
    2. Почему в обучении без учителя не бывает однозначно «правильного» ответа?
    3. Назови три главных класса задач без таргета и что объединяет их все.
    4. Какая общая проблема возникает из-за отсутствия меток $y$?

??? success "Ответы"

    1. В supervised есть метки $y_i$ и метрика ошибки; в unsupervised меток нет, есть только признаки $x_i$, и модель ищет структуру в распределении $p(x)$.
    2. Никто не задал, по какому признаку считать объекты похожими; разные алгоритмы дают разные, но одинаково валидные разбиения. Ответ бывает полезным или бесполезным, а не правильным или неправильным.
    3. Кластеризация (разбить на группы), снижение размерности (отображение в меньшую размерность с сохранением структуры), поиск аномалий (нетипичные точки с малой $p(x)$). Объединяет отсутствие $y$.
    4. Проблема валидации: нет таргета — нет accuracy, и непонятно, чем мерить качество результата.
