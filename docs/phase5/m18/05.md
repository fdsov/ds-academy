# M18.5 · Многослойный перцептрон и forward pass

### Интуиция

MLP — это стопка полносвязных (fully connected, dense) слоёв. «Полносвязный» значит: каждый нейрон слоя соединён с каждым выходом предыдущего. Данные текут вперёд: вход превращается в скрытое представление, оно — в следующее, и так до выхода. Этот прямой проход и называется forward pass. Каждый скрытый слой строит всё более абстрактные признаки: первый ловит простые комбинации (активность × размер ставки), глубже — сложные сегменты поведения.

### Формально

Обозначим $a^{(0)} = x$. Для слоя $l = 1, \dots, L$:

$$
z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}, \qquad a^{(l)} = \phi^{(l)}(z^{(l)})
$$

где $W^{(l)}$ имеет размер (нейронов в слое $l$) × (нейронов в слое $l-1$). Выход $a^{(L)}$ — предсказание. Для бинарной классификации $\phi^{(L)} = \sigma$ и один выход; для $K$ классов — softmax:

$$
\mathrm{softmax}(z)_k = \frac{e^{z_k}}{\sum_{j=1}^{K} e^{z_j}}
$$

!!! question "Проверь себя"

    1. Что значит «полносвязный» (dense) слой?
    2. Запиши два уравнения, описывающих один слой forward pass, и поясни, что такое $z^{(l)}$ и $a^{(l)}$.
    3. Какой размер у матрицы $W^{(l)}$ и почему именно такой?
    4. Какую активацию выходного слоя выбирают для бинарной классификации, а какую — для $K$ классов?

??? success "Ответы"

    1. Каждый нейрон слоя соединён с каждым выходом предыдущего слоя.
    2. $z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}$ — линейная (аффинная) часть; $a^{(l)} = \phi^{(l)}(z^{(l)})$ — активация. $z^{(l)}$ — предактивация, $a^{(l)}$ — выход слоя.
    3. (нейронов в слое $l$) × (нейронов в слое $l-1$) — чтобы умножение на вектор $a^{(l-1)}$ давало вектор размерности слоя $l$.
    4. Для бинарной — сигмоида с одним выходом; для $K$ классов — softmax по $K$ выходам.

!!! example "Мини-задание"

    Сеть: вход 4 признака, скрытый слой 8 нейронов, выход 1. Выпиши размеры $W^{(1)}$, $b^{(1)}$, $W^{(2)}$, $b^{(2)}$ и проверь по цепочке размерностей, что forward pass на одном примере даёт скаляр на выходе. (Ответ: $W^{(1)}$ 8×4, $b^{(1)}$ 8, $W^{(2)}$ 1×8, $b^{(2)}$ 1.)
