# M18.9 · Инициализация весов

### Интуиция

С чего начинать веса? Не с нулей — тогда все нейроны слоя идентичны, получают одинаковый градиент и навсегда остаются клонами (проблема симметрии). Не со слишком больших значений — активации насытятся, градиенты взорвутся или затухнут. Нужна случайная инициализация с правильным масштабом, чтобы сигнал и градиент проходили через десятки слоёв, не угасая и не взрываясь.

### Формально

**Xavier/Glorot** — для сигмоиды/tanh: дисперсия весов $\mathrm{Var}(W) = 2 / (n_{in} + n_{out})$. **He** — для ReLU (учитывает, что ReLU обнуляет половину): $\mathrm{Var}(W) = 2 / n_{in}$. PyTorch инициализирует `nn.Linear` разумно по умолчанию, но для нестандартных архитектур инициализацию задают явно через `torch.nn.init`.

!!! question "Проверь себя"

    1. Почему нельзя инициализировать все веса нулями?
    2. Чем грозит инициализация слишком большими значениями?
    3. В чём разница между Xavier/Glorot и He, и под какие активации каждая?

??? success "Ответы"

    1. Тогда все нейроны слоя идентичны, получают одинаковый градиент и навсегда остаются клонами — проблема симметрии, сеть не учится.
    2. Активации насыщаются, а градиенты взрываются или затухают, и сигнал не проходит через слои.
    3. Xavier/Glorot: $\mathrm{Var}(W) = 2/(n_{in}+n_{out})$ — для сигмоиды/tanh. He: $\mathrm{Var}(W) = 2/n_{in}$ — для ReLU, учитывает, что ReLU обнуляет половину входов.
