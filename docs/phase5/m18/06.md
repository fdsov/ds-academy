# M18.6 · Функции потерь

### Интуиция

Лосс — это число, которое говорит, насколько сильно сеть ошиблась. Чем оно меньше, тем лучше предсказание. Обучение — это минимизация лосса по весам. Выбор лосса диктуется задачей: для регрессии (предсказать LTV в деньгах) штрафуем за квадрат отклонения; для классификации (депозит/нет) — за уверенность в неправильном ответе.

### Формально

**MSE** (среднеквадратичная ошибка) для регрессии:

$$
\mathcal{L}_{\text{MSE}} = \frac{1}{N} \sum_{i=1}^{N} (y_i - \hat{y}_i)^2
$$

**Бинарная кросс-энтропия** для классификации:

$$
\mathcal{L}_{\text{BCE}} = -\frac{1}{N} \sum_{i=1}^{N} \left[ y_i \log \hat{y}_i + (1 - y_i) \log(1 - \hat{y}_i) \right]
$$

Интуиция кросс-энтропии: если истинный ответ 1, а сеть выдала $\hat{y} = 0.99$, штраф $-\log 0.99 \approx 0.01$ крошечный; если выдала $0.01$ — штраф $-\log 0.01 \approx 4.6$ огромный. Уверенная ошибка наказывается жёстко.

!!! question "Проверь себя"

    1. Что такое лосс и как через него формулируется цель обучения?
    2. Какой лосс берут для регрессии, а какой — для бинарной классификации?
    3. Почему кросс-энтропия жёстко наказывает уверенную ошибку? Поясни на числах.

??? success "Ответы"

    1. Лосс — число, показывающее, насколько сеть ошиблась. Обучение — минимизация лосса по весам.
    2. Регрессия — MSE (среднеквадратичная ошибка); бинарная классификация — бинарная кросс-энтропия (BCE).
    3. Штраф равен $-\log \hat{y}$ для истинного класса. При $\hat{y}=0.99$ штраф $\approx 0.01$, а при $\hat{y}=0.01$ (уверенно неверно) $\approx 4.6$ — логарифм взрывается у нуля.

!!! example "Мини-задание"

    Истинный ответ $y=1$. Посчитай BCE-штраф для предсказаний $\hat{y} = 0.9$, $0.5$ и $0.1$ по формуле $-\log \hat{y}$. Убедись, что штраф растёт по мере того, как сеть всё увереннее ошибается.
