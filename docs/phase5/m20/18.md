# M20.18 · Ресурсы

- Jurafsky, Martin. "Speech and Language Processing" (3-е изд., черновики обновляются) — фундамент по эмбеддингам, токенизации, языковым моделям.
- Anthropic, "Building Effective Agents" и docs по tool use / MCP — практика агентов и инструментов от первоисточника.
- OpenAI Cookbook (RAG, эмбеддинги, structured outputs, evals) — рабочие рецепты под актуальный API.
- RAGAS docs — метрики и методология оценки RAG-систем (faithfulness, answer/context relevance).
- Rafailov et al., "Direct Preference Optimization" (2023) — первоисточник по DPO, чтобы понять современное выравнивание.
- Lewis et al., "Retrieval-Augmented Generation" (2020) — оригинальная статья про RAG.
- Курс DeepLearning.AI "Building and Evaluating Advanced RAG" — короткий практический, про reranking и оценку.
- Документация Qdrant и pgvector — стандартные векторные хранилища 2026, паттерны фильтрации и ANN-поиска.
