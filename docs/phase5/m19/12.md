# M19.12 · Ресурсы

- Goodfellow, Bengio, Courville — Deep Learning (главы про CNN и RNN; теоретический фундамент, бесплатно онлайн).
- Vaswani et al. — Attention Is All You Need (оригинальная статья про трансформер; читать в первоисточнике обязательно).
- Jay Alammar — The Illustrated Transformer и The Illustrated Word2Vec (лучшие визуальные объяснения внимания).
- Hugging Face — NLP Course и LLM Course (бесплатные, постоянно обновляются, практика с актуальным API).
- Andrej Karpathy — серия Neural Networks: Zero to Hero и nanoGPT (сборка трансформера с нуля, видео + код).
- Stanford CS231n — Convolutional Neural Networks for Visual Recognition (классический курс по CNN, конспекты в открытом доступе).
- Lilian Weng — блог lilianweng.github.io, разборы attention, diffusion и архитектур (глубоко и строго).
- Dive into Deep Learning (d2l.ai) — интерактивный учебник с кодом на PyTorch, регулярно обновляется.
