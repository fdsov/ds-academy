# M21.10 · Ресурсы

- Anthropic, документация Claude Code и prompt engineering guide (актуальная версия 2026) — паттерны агентной работы и промптинга.
- Andriy Burkov, «The Hundred-Page Language Models Book» (2025) — как устроены LLM внутри, почему галлюцинируют.
- Документация polars (latest) и scikit-learn (latest) — первоисточник для сверки API против галлюцинаций.
- Ron Kohavi, Diane Tang, Ya Xu, «Trustworthy Online Controlled Experiments» — чтобы самому понимать статистику A/B и ловить ошибки AI.
- Статья «On the Dangers of Stochastic Parrots» (Bender et al.) — почему правдоподобие текста не равно истинности.
- GDPR и гайдлайны локального регулятора азартных игр по обработке персональных данных — обязательно для compliance.
- Chip Huyen, «AI Engineering» (2025) — как встраивать LLM в продуктовые и аналитические воркфлоу инженерно.
