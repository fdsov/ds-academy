# M1.15 · Ресурсы

- Официальная документация uv (docs.astral.sh/uv) — единственный источник истины по командам и lockfile, актуален на 2026.
- Документация marimo (docs.marimo.io) — концепция реактивных ноутбуков и проблема скрытого состояния, с примерами.
- Cookiecutter Data Science (cookiecutter-data-science.drivendata.org) — обоснование структуры DS-проекта и принципа неизменяемых сырых данных.
- Pro Git (Scott Chacon, git-scm.com/book, бесплатно) — фундаментальная книга по git, главы 1-3 обязательны.
- The Twelve-Factor App (12factor.net), фактор III «Config» — почему конфигурация и секреты живут в окружении, не в коде.
- Документация CPython по venv и What's New in Python 3.14 (docs.python.org) — про окружения и про официально поддерживаемый free-threaded-билд и JIT.
- Статьи о воспроизводимости Jupyter-ноутбуков (например, исследования Pimentel et al. о доле невоспроизводимых публичных ноутбуков) — эмпирическая база проблемы скрытого состояния.
- Документация ruff (docs.astral.sh/ruff) — современный линтер и форматтер, заменивший связку black/isort/flake8.
