# M2.2 · Что ты освоишь

- [ ] Объяснять, чем изменяемые (mutable) типы отличаются от неизменяемых (immutable) и как это связано с моделью «имя — это ярлык на объект»
- [ ] Выбирать правильную коллекцию (list / tuple / dict / set) под задачу по сложности операций, а не наугад
- [ ] Писать comprehensions и генераторы, понимая, когда ленивые вычисления экономят память на гигабайтных логах
- [ ] Проектировать функции с args/kwargs, аннотациями типов и без классической ловушки изменяемых дефолтов
- [ ] Применять ООП там, где оно реально упрощает DS-код (трансформеры, конфиги, кастомные модели), и не плодить классы там, где хватит функции
- [ ] Читать и писать декораторы и контекстные менеджеры, понимая, что под капотом
- [ ] Уверенно использовать стандартную библиотеку для DS: `collections`, `itertools`, `datetime`, `pathlib`, `json`
- [ ] Писать питоничный код по PEP 8, который пройдёт ревью без двадцати комментариев
